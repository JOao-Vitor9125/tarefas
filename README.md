Código simples com implementação do padrão de projeto  FACADE. Um sisteminha de check-in e check-out de hóspedes de um hotel por meio da recepção. 

O padrão de projeto FACADE é muito interessante e eficianete na nossas opinião. Ele pode ser usado em meio a uma arquitetura agindo como um bloco onteiro para uma função específica onde é possivel saber em qual bloco procurar um problema, sabendo qual funcionalidade está com problema, visto que um conjunto de classes são utilizadas em conjunto por uma classe que as contêm.

Alunos: João Vitor Adiers Xavier / Iran Matos Barbosa.

Instalação e execução: Apenas baixe o arquivo e rode tendo todas as dependecias pardão do typescript.
Descomente a linha comentada no código para ver o outro lado da funcionalidade.
